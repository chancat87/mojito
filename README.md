<p align="center">Mojito是一个低代码，可视化，可扩展开发平台</p>

##  🔥特性

- 点选、拖拽、缩放的可视化操作
- 群组/解散、撤销/重做、图层显示/隐藏、锁定/解锁、对齐和排序
- 支持使用React和Vue3自定义组件，样式隔离
- 组件样式可视化配置
- 组件间通讯、页面下钻、动态脚本...
